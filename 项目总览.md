## 🔑 关键文件说明

### 核心代码文件

#### 1. demo_simple.py
**作用**: 最简单的多Agent演示

**核心代码**:
```python
# 创建两个Agent
pm_agent = AssistantAgent(name="Product_Manager", ...)
developer_agent = AssistantAgent(name="Developer", ...)

# 启动对话
user_proxy.initiate_chat(pm_agent, message="任务描述")
```

**修改点**:
- `task`变量: 修改任务描述
- `system_message`: 修改Agent角色
- `llm_config`: 修改LLM配置

#### 2. demo_langgraph.py
**作用**: 完整的瀑布开发流程

**核心代码**:
```python
# 定义状态
class TeamState(TypedDict):
    messages: list
    current_stage: str
    ...

# 定义Agent函数
def pm_agent(state):
    # PM的逻辑
    ...

# 构建流程图
workflow = StateGraph(TeamState)
workflow.add_node("PM", pm_agent)
workflow.add_edge("PM", "Requirements")

# 运行
app = workflow.compile()
result = app.invoke(initial_state)
```

**修改点**:
- `user_request`: 修改需求
- Agent函数: 修改每个Agent的逻辑
- `should_continue`: 修改流程控制

### 配置文件

#### .env
**作用**: 存储API密钥和配置

**内容**:
```bash
OPENAI_API_KEY=sk-你的密钥
MODEL_NAME=gpt-3.5-turbo
```

**重要**:
- 不要提交到Git（包含密钥）
- 不要分享给他人
- 参考.env.example创建

#### requirements.txt
**作用**: 定义依赖包

**使用**:
```bash
# 安装所有依赖
pip install -r requirements.txt

# 只安装最小依赖
pip install pyautogen openai python-dotenv
```
