## 4. Bug列表

### 4.1 Bug详情

#### Bug-001: 盖房条件错误
- **严重程度**: P1
- **功能需求**: FR-008
- **核心规则**: 只能在停着的地产上盖房
- **问题描述**: 路过自己的地产也能盖房
- **复现步骤**:
  1. 玩家拥有地产X
  2. 掷骰子路过地产X
  3. 检查是否显示"盖房"选项
- **实际结果**: 显示"盖房"选项
- **预期结果**: 不显示"盖房"选项
- **证据**: [截图/日志]
- **复现概率**: 100%

#### Bug-002: 卡牌未循环
- **严重程度**: P1
- **功能需求**: FR-009
- **核心规则**: 卡牌队列循环
- **问题描述**: 卡牌用完不再出现
- **复现步骤**:
  1. 移动到机会格
  2. 拿取队首卡牌
  3. 再次移动到机会格
  4. 检查是否拿到相同卡牌
- **实际结果**: 不会拿到相同卡牌
- **预期结果**: 应该循环使用，执行后移至队尾
- **证据**: [截图/日志]
- **复现概率**: 100%

#### Bug-003: 破产处理不完整
- **严重程度**: P0
- **功能需求**: FR-014
- **核心规则**: 破产前先变卖资产
- **问题描述**: 未实现递归变卖资产流程
- **复现步骤**:
  1. 玩家资金不足
  2. 检查是否显示变卖资产对话框
- **实际结果**: 直接判定破产
- **预期结果**: 应该先变卖资产（房子半价、土地半价抵押）
- **证据**: [截图/日志]
- **复现概率**: 100%
